## **Design System Foundation**

**Colors:**
```css
--bg-primary: #0a0a0a;
--bg-secondary: #111111;
--accent-leek: #4ade80; /* Neon green */
--accent-leek-glow: #86efac;
--accent-cyan: #22d3ee;
--text-primary: #f0fdf4;
--text-secondary: rgba(240,253,244,0.7);
--danger: #f87171;
--warning: #fbbf24;
```

**Typography:** JetBrains Mono (400,600) for code elements, Inter (400,600,700,900) for UI. Headings: 5xl-8xl.

**Animations:**
```css
/* Code typewriter */
@keyframes code-type { 0% { opacity: 0 } 5% { opacity: 1 } 100% { opacity: 1 } }

/* Leek glow pulse */
@keyframes leek-glow { 0%,100% { text-shadow: 0 0 10px var(--accent-leek) } 50% { text-shadow: 0 0 20px var(--accent-leek), 0 0 30px var(--accent-leek) } }

/* Scroll velocity */
.reveal { opacity: 0; transform: translateY(30px) }
.reveal.active { opacity: 1; transform: translateY(0); transition: 0.8s cubic-bezier(0.16, 1, 0.3, 1) }

/* Glitch (for loading states) */
@keyframes glitch { 0% { transform: translate(0) } 20% { transform: translate(-2px, 2px) } 40% { transform: translate(-2px, -2px) } 60% { transform: translate(2px, 2px) } 80% { transform: translate(2px, -2px) } 100% { transform: translate(0) } }
```

**Icon Library:** Iconify Solar (duotone) + Iconify MingCute (for social). Use pixel-perfect 24px grid.

## **Technical Requirements**

1. **Single HTML file** - All pages in DOM, switch with `data-page` attribute
2. **No frameworks** - Vanilla JS only, <50KB total
3. **Iconify CDN** - `https://cdn.jsdelivr.net/npm/@iconify/iconify@3.0.0/dist/iconify.min.js`
4. **Dark mode only** - System dark mode detection
5. **Performance** - Critical CSS inline, non-critical at bottom
6. **Accessibility** - ARIA labels, keyboard navigation (Tab, Enter, Escape), focus traps in modals
7. **Mobile** - Bottom navigation bar, swipe gestures (left/right to switch pages)
8. **Hotkeys** - `Cmd+K` search, `Cmd+N` new capture, `?` help overlay
9. **Animations** - Reduced motion support: `@media (prefers-reduced-motion: reduce) { * { animation: none !important } }`
10. **Deploy** - Drop in Replit/Aura, runs on any static host
